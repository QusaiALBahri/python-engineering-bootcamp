# 📝 Day 11: Web Development & AI - EXERCISES

## Exercise 1: Build a Simple Flask App
Create a Flask app with:
- Home route returning HTML
- `/greet/<name>` route
- `/api/data` POST endpoint returning JSON
- Error handling for 404

## Exercise 2: TODO API
Build a RESTful API for managing todos:
- GET /todos - list all
- POST /todos - create
- PUT /todos/<id> - update
- DELETE /todos/<id> - remove

## Exercise 3: AI Chatbot
Create a Flask app with AI integration:
- Chat endpoint accepting user messages
- Connect to Groq/OpenAI API
- Return AI responses
- Store conversation history

## Exercise 4: Weather API Wrapper
Build a Flask wrapper that:
- Accepts city name queries
- Returns JSON with weather info
- Uses OpenWeather API (free tier)

## Exercise 5: Challenge - Full Chat Application
Build a complete web app with:
- Flask backend
- HTML/CSS frontend
- AI chatbot integration
- Message history display
- Clean UI with styling
